# MiniCore Plugin - Bağışçılara Özel

Geliştirici DrMalito: v1.3.2 Sürümü

## Plugin Açıklamaları

* **MiniTrain:** Tren modülü özelliği.
* **PetLinkBeta:** DP-Link kullanabilmenizi sağlar. Slot-bar veya pet ekranından seçer.
* **SmartFormation:** En iyi droid düzenini seçer.
* **SmartAbilities:** Gemi yeteneklerini otomatik olarak kullanır.
