# TYX Plugin

Geliştirici TeYroX: v1.2.6 Sürümü

> 📷 *TYX plugin genel görseli buraya eklenecek*

## Plugin Açıklamaları

* **Cubi Module:** Kendi şirket haritanızda cubikonun konumunda bekleyerek kesmenizi sağlar.
* **NPCRadiusList:** Bazı NPC'lerin mesafelerini otomatik olarak ayarlayan özellik.
* **BOXList:** Hammaddeler, Ganimet Anahtarları, Avantaj Kutuları ve Özel Etkinliklerde gelen kutuları toplama listesine ekler.
