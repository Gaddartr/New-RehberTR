# Lean Plugin - Bağışçılara Özel

Geliştirici leanon00: v1.3.1 Sürümü

## Plugin Açıklamaları

* **New Stats Task:** İstatistikleri görmeniz ve sıfırlamanız için ana menüye bir buton ekler.
* **New Discord Task:** Discordunuza bot istatistiklerini göndermenizi sağlar.
* **Npc Stats:** NPC statslarını "New Stats Task'a" ekler. *(Bot dilini İngilizce yapın)*
* **GG stats:** Spinner statslarını "New Stats Task'a" ekler.
* **Ambulance [Beta]:** Aegis ve Solace yeteneğini kullanmanızı sağlar.
* **QZone Extra:** Belirli şartlarda karantina kapısından çıkmanızı sağlar.
* **GroupEnforcer [Beta]:** Grupta belirlediğiniz sayıda oyuncu kaldığında Tag değiştirmenizi sağlar.
* **EquipmentStats Beta [Donor]:** Ekipman istatistiklerini görmenizi sağlar.
* **TempReturnWindowsFix:** Geri dönen giriş etkinliği penceresini kapatmayı sağlar.

{% hint style="info" %}
TempReturnWindowsFix pencere kapanmıyorsa bot çözünürlüğünü 1600x1000 yapmayı deneyin.
{% endhint %}
