# Pika Plugin - Bağışçılara Özel

Geliştirici PikaPika: v1.6.72 Sürümü

Discord: https://discord.gg/7CuG4wqD5r

## Plugin Açıklamaları

* **Auction:** Açık artırmadaki ürünlere belirlediğiniz dakikada ve miktarda teklif vermenizi sağlar. (**Captcha sisteminden dolayı stabil çalışmıyor**)
* **Dispatch (Donor):** Sevkiyat özelliğine istediğiniz robotları otomatik olarak gönderip toplamanızı sağlar.
  * **Hire:** Hangi sevkiyat robotunu göndermek istiyorsanız onu seçin.
  * **Use Prime:** Prime kuponu kullanarak öğeleri hızlı toplar.
  * **Build Amount 0** yazarsanız özellik devre dışı kalır.
* **Hitac Module Pika:** PvP haritalarındaki Hitac NPC'sini takip edip öldürmenizi sağlar.
* **Labyrinth Config Changer:** Labirent haritasından önce/sonra istediğiniz konfigürasyonla çalışmaya devam eder.
* **Escort Config Changer:** Eskort haritasından önce/sonra istediğiniz konfigürasyonla çalışmaya devam eder.
