# Default Plugin

Geliştirici Popcorn: v1.7.4 Sürümü

> 📷 *Default plugin genel görseli buraya eklenecek*

## Plugin Açıklamaları

* **Captcha Picker:** Oyun içerisindeki Kara ve Kırmızı kutuları otomatik olarak geçmenizi sağlar. Botunuz yada Default Plugininiz güncel değilse kutuları toplayamaz.
* **Palladium module:** Palladium toplayıp otomatik olarak harita değiştirerek (5-3 > 4-4 > x-5 > 4-5 > 5-1 > 5-2 > 5-3) satmanızı sağlar.
* **Flash patcher:** Güncelleme ile pasif duruma gelen Flash Player'ın tekrar aktif olmasını sağlar.
* **Do client updater:** Oyun istemcisi ile botunuzun aynı sürümde olmasını sağlar.
* **Sample module:** Plugin ve modül yazmak isteyenler için öğreticidir. Herhangi bir işlevi yoktur.
