# Kekka Plugin - Bağışçılara Özel

Geliştirici Punisher, Popcorn, Ace: v1.6.6 beta 21 Sürümü

Discord: https://discord.gg/m4CMbBd5h6

## Plugin Açıklamaları

* **Galaxy module:** Çoğu galaksi kapısını yapar ve kamikaze kullanarak kapıları bitirebilirsiniz.
* **GoP Module:** Gauntlet of Plutus etkinlik kapısını yapmanızı sağlar.
* **LoW module:** x-3 haritanızda bulunan LoW kapısını yapmanızı sağlar.
* **Eternal Blacklight:** x-BL haritasındaki Ebedi kapıyı yapmanızı sağlar.

{% hint style="info" %}
Güveyle NPC öldürürken kalkanınız 0 oluyorsa; botun genel ayarlarından ve Eternal modülünden kalkan tamir yüzdesini 0 yapın.
{% endhint %}

* **Hades module:** Hades kapısını yapmanızı sağlar.
* **Guardian [KD]:** Grup liderini takip eder ve saldırdığı aynı hedefe saldırır. *(Kekka Donor)*
* **Chaos Spawner [KD]:** Protegit tantanası etkinliğinde CPU'ları otomatik basmanızı sağlar. *(Kekka Donor)*
* **Galaxy Helper:** Galaxy kapıları hakkında istatistik sağlar.
* **Skylab Builder:** Gerekli hammaddeler varsa skylabınızı 20. seviyeye otomatik geliştirir.
